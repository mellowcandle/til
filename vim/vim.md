# VIM

## Misc

1. **Reload .vimrc after change:**
	`:so % `(while it's in buffer)
2. **Lookup word under cursor MAN page:**
	`:K` (when cursor on word)

3. **Jump to enclosing function:**
	`[[`

4. **Search for complete word:**
 	`/\<word\>`
5. **Search and replace with confirmation:**
	`:%s/old/new/gc`
